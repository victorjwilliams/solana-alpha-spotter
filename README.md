## Overview

Solana Alpha Spotter is a front-end web application that connects to live Solana blockchain data to help users discover new token launches early. It tracks wallet activity, on-chain signals, and token momentum — all in a clean, readable interface built for speed.

---

## Features

- 🚀 **New token detection** — surfaces recently launched tokens on the Solana network
- 📡 **Live blockchain data feeds** — pulls real-time on-chain information via REST APIs
- 👛 **Wallet activity tracking** — monitors notable wallet movements tied to new tokens
- 📊 **Momentum indicators** — scores tokens based on early activity signals
- ⚡ **Fully client-side** — no backend required, fast load times, easy to deploy
- 🛡️ **Responsible design** — includes risk awareness context for all data displayed

## Getting Started

```bash
# Clone the repository
git clone https://github.com/victorjwilliams/solana-alpha-spotter.git

# Navigate into the project
cd solana-alpha-spotter

# Open in your browser
open index.html
```

No installation or build step needed. Runs entirely in the browser.

---

## How It Works

1. On load, the app queries Solana network data via public REST endpoints
2. New token listings are parsed and ranked by activity score
3. Wallet movement data is layered in to surface tokens with notable interest
4. Results are displayed in a clean card UI with key metrics per token
